# SEO Wave 3 Scale Plan

This document captures the scale-stage roadmap after wave 1 and wave 2 are stable.

## 1) Long-tail scale

- Expand topic hubs with high-intent long-tail pages:
  - `best chapter for anxiety`
  - `gita verse for focus`
  - `how to apply karma yoga at work`
- Keep one primary intent per page.
- Reuse verse/chapter references from `gitaData.json` to avoid thin pages.

## 2) Localization strategy

- Prioritize language landing pages only where meaningful depth exists.
- Candidate languages from demand:
  - Hindi
  - Tamil
  - Telugu
  - Marathi
  - Kannada
  - Gujarati
- Do not publish placeholder language pages.

## 3) Performance + crawl maturity

- Improve template splitting/lazy loading for SEO-critical pages.
- Keep render path stable for chapter and verse routes.
- Maintain sitemap freshness discipline (content mtime strategy already in place).

## 4) Continuous optimization cycle

Monthly:
- review keyword movement by intent cluster
- add internal links from topic hubs to newly optimized verses
- update chapter metadata for rising search themes

Quarterly:
- revise hub taxonomy based on Search Console query growth
- add or merge hub pages to avoid cannibalization
